Adapter для преобразования к определённому интерфейсу.

Bridge разделяет абстракцию от ее реализации, так что они могут изменяться независимо друг от друга.

Composite позволяет обрабатывать группы объектов так, как будто это единичный объект.
1. Java: Древовидные структуры данных
2. Spring: Иерархия бинов
3. Hibernate: Иерархия объектов в БД
4. Графический интерфейс Java (AWT и Swing): Иерархия компонентов

Decorator динамически добавляет новые обязанности объекту.
1. Java I/O Streams: BufferedInputStream может быть обернут в DataInputStream

Facade предоставляет простой интерфейс для сложной системы классов, библиотек и других объектов.
1. Java утилиты: java.util.Calendar, java.net.URL
2. Spring JdbcTemplate
3. Hibernate Session

Flyweight позволяет эффективно использовать большое количество мелких объектов, используя разделяемые объекты.
1. String pool и другие пулы. Для повторного использования объектов
2. Hibernate кэш ???

Proxy контролирует доступ к объекту, добавляя дополнительную логику к его операциям.
1. Hibernate ленивая загрузка
2. @Transactional
3. AOP для добавления advices
