Builder используется для создания объектов с составными частями (комплексными объектами), позволяя создавать различные конфигурации объектов, скрывая детали их создания.
1. StringBuilder, StringBuffer
2. Hibernate: CriteriaBuilder
3. Spring Framework: RestTemplateBuilder

Singleton гарантирует, что класс имеет только один экземпляр, и предоставляет глобальную точку доступа к этому экземпляру.
1. Spring Framework: Singleton bean
2. Java: Enum

Prototype используется для создания новых объектов путем клонирования существующих объектов, вместо создания новых объектов "с нуля".
1. Spring Framework: Prototype bean
2. Java: Clone()

Factory Method определяет общий интерфейс для создания объектов, но позволяет подклассам решать, какой класс инстанцировать. Таким образом, создание объектов происходит через метод-фабрику.
1. Java создание коллекций: ArrayList::new, HashMap::new, HashSet::new ... являются фабричными методами
2. Hibernate: SessionFactory::openSession являются фабричными методам
