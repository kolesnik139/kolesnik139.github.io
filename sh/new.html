Виртуальные потоки
Обычный поток (Platform Thread)
Это обёртка над реальным потоком ОС
1 Java-поток = 1 поток операционной системы
Планируется (scheduler) операционной системой
Дорогой по памяти и созданию

Виртуальный поток (Virtual Thread)
Не является отдельным потоком ОС
Управляется JVM
Планируется внутренним scheduler'ом JVM
Выполняется поверх небольшого числа реальных потоков

Sequenced Collections 
    interface SequencedCollection<E> extends Collection<E> {
        // new method
        SequencedCollection<E> reversed();
        // methods promoted from Deque
        void addFirst(E);
        void addLast(E);
        E getFirst();
        E getLast();
        E removeFirst();
        E removeLast();
    }

    interface SequencedSet<E> extends Set<E>, SequencedCollection<E> {
        SequencedSet<E> reversed();    // covariant override
    }

    interface SequencedMap<K,V> extends Map<K,V> {
        // new methods
        SequencedMap<K,V> reversed();
        SequencedSet<K> sequencedKeySet();
        SequencedCollection<V> sequencedValues();
        SequencedSet<Entry<K,V>> sequencedEntrySet();
        V putFirst(K, V);
        V putLast(K, V);
        // methods promoted from NavigableMap
        Entry<K, V> firstEntry();
        Entry<K, V> lastEntry();
        Entry<K, V> pollFirstEntry();
        Entry<K, V> pollLastEntry();
    }

Pattern Matching в switch
Теперь это полноценный механизм с поддержкой разных типов и улучшенной проверкой.
    String describe(Object obj) {
        return switch (obj) {
            case Integer i -> "Число " + i;
            case String s -> "Строка " + s;
            case null -> "null";
            default -> "Неизвестно";
        };
    }
