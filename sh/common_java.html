При переопределении (override) методов:
1. Типы параметров: == такие же
2. Возвращаемый тип: <= такой же, или подтип (ковариантный возвращаемый тип).
3. Уровень доступа: >= такой же, или шире.
4. Исключение: <= такое же, или уже.

Если в интерфейсах A и B есть метод default void foo(){} 
    public class C implements A, B {
    @Override
    public void foo() {
        A.super.foo();
    }
    }
