Immutable Types

1. String: Все операции над строками создают новые строки.
2. Integer, Long, Short, Byte, Character, Double, Float, и т. д.: Оболочки для примитивных типов неизменяемы.
3. BigInteger и BigDecimal
4. LocalDate, LocalTime, LocalDateTime, Instant: Классы из пакета java.time
5. Collections.unmodifiableXXX(): C помощью методов Collections.unmodifiableXXX() создаётся Immutable обёртка
6. Enum: Типы-перечисления
7. java.lang.StackTraceElement

Immutable POJO

    public class Human {
        private final String firstName;
        private final String lastName;
        private final List<Human> relatives;

        public Human(String firstName, String lastName, List<Human> relatives) {
            this.firstName = firstName;
            this.lastName = lastName;
            // Создаем защитную копию списка relatives
            this.relatives = new ArrayList<>(relatives);
        }

        public String getFirstName() {
            return firstName;
        }

        public String getLastName() {
            return lastName;
        }

        public List<Human> getRelatives() {
            // Возвращаем защитную копию списка relatives
            return new ArrayList<>(relatives);
        }
    }
